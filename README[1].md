
# Wildfire Mask NFC Tool

This folder is ready for GitHub Pages deployment.

## How to Publish
1. Go to GitHub → Create repository named **wildfire-mask**
2. Upload all files from this ZIP
3. Go to Settings → Pages
4. Select "Deploy from main branch"
5. Your NFC tool will appear at:
   https://holmesau.github.io/wildfire-mask/

## NFC Requirements
- Android phone
- Chrome browser
- HTTPS (GitHub Pages provides this)
